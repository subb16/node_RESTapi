MathonGo API

Welcome to the Chapter Performance Dashboard API made by Subhash Patel

Available Endpoints

GET /api/v1/chapters

Retrieve all chapters with optional filters

curl https://mathongo-restapi.onrender.com/api/v1/chapters

GET /api/v1/chapters/:id

Retrieve any chapter by its id

curl https://mathongo-restapi.onrender.com/api/v1/chapters/:id

POST /api/v1/chapters

Upload new chapters (Admin only)

curl -X POST -H "x-admin-token: YOUR_TOKEN" -F "file=@data.json" https://mathongo-restapi.onrender.com/api/v1/chapters
